# Wazuh-Based SOC and Threat Monitoring Lab

## Project Overview
This project demonstrates a Security Operations Center (SOC) lab setup using Wazuh SIEM.

## Technologies Used
- Kali Linux
- Wazuh
- Windows 11
- VirtualBox

## Features
- Endpoint Monitoring
- File Integrity Monitoring (FIM)
- Real-Time Alert Detection
- Security Event Monitoring
- Threat Hunting Dashboard

## Architecture
Windows Agent ? Wazuh Manager (Kali Linux) ? Dashboard Monitoring
